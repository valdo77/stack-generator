# Stack Generator

Stack generator is a CLI tool that allows you to generate an application template in one command.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install stack generator.

```bash
pip install stack-gen
```

## Usage

```bash
sg --template=fastapi
sg --template=vuejs
```
